## Phần 2: Cấu hình trên máy Windows để SSH ### Bước 1: Cài cloudflared trên Windows Tải và cài từ PowerShell (chạy với quyền Admin): 
powershell winget install --id Cloudflare.cloudflared ``` Hoặc tải trực tiếp: https://github.com/cloudflare/cloudflared/releases/latest → file `cloudflared-windows-amd64.msi` ### Bước 2: Cấu hình SSH config Mở file `C:\Users\<TenBan>\.ssh\config` (tạo mới nếu chưa có) và thêm: ``` Host ssh.openinfra.space HostName ssh.openinfra.space User bobaonhan123 ProxyCommand cloudflared access ssh --hostname %h ``` ### Bước 3: Kết nối SSH Mở **Terminal / PowerShell / CMD** và chạy: ```bash ssh ssh.openinfra.space ``` Hoặc nếu không muốn cấu hình SSH config, dùng trực tiếp: ```bash ssh -o ProxyCommand="cloudflared access ssh --hostname ssh.openinfra.space" bobaonhan123@ssh.openinfra.space ``` ### Lưu ý quan trọng - **Không cần** login cloudflared trên Windows (không cần `cloudflared login`) - **Không cần** mở port trên server — mọi thứ đi qua Cloudflare - Lần đầu kết nối sẽ hỏi xác nhận host key → gõ `yes` - Nếu dùng **VS Code Remote SSH**, cũng chỉ cần thêm config ở trên vào SSH config là dùng được ### Nếu muốn dùng SSH key (khuyến nghị) Trên Windows, tạo key: ```powershell ssh-keygen -t ed25519 ``` Rồi copy public key lên server (chạy trên Windows): ```powershell type $env:USERPROFILE\.ssh\id_ed25519.pub | ssh ssh.openinfra.space "cat >> ~/.ssh/authorized_keys"
